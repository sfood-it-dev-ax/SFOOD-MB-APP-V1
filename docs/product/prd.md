# Product Requirements (Backend Extract)

## Product
Post-it Board personal memo web app backend APIs.

## Iteration 01 Objective
Deliver core API skeleton and CRUD capabilities for:
- Session-based authentication endpoints.
- Board lifecycle APIs.
- Memo lifecycle APIs.
- Memo type lookup API.

## Out of Scope
- Frontend implementation.
- Real Google token verification integration.
- File upload/search/import/export.
